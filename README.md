# ATM Simulator - UAS PBO

Aplikasi simulasi ATM berbasis Python menggunakan Tkinter
yang menerapkan konsep Object-Oriented Programming (OOP).

## Fitur
- Login dan Registrasi
- Setor Tunai
- Tarik Tunai
- Transfer Antar Rekening
- Cek Saldo
- Riwayat Transaksi

## Konsep OOP
- Encapsulation
- Inheritance
- Polymorphism (Method Overriding)
